# goit-pycore-hw-04
# Author: Maksym Kaniuka

Домашня робота 4: Обробка файлів, інструменти CLI та консольний бот.

## Завдання
**task1.py - total_salary** — Зчитує файл із зарплатами та повертає загальну суму + середню зарплату.
**task2.py - get_cats_info** — Розбирає файл із даними cat у список словників.
**task3.py - Візуалізатор каталогів** — CLI-скрипт, який друкує дерево каталогів з кольоровим кодуванням за допомогою colorama.
**task4.py - Бот-помічник** — Інтерактивний консольний бот для керування телефонною книгою (додати, змінити, телефон).

## Використання
**Завдання 1**
python task1/task1.py

**Завдання 2**
python task2/task2.py

**Завдання 3**
pip install colorama
python task3/task3.py /шлях/до/каталогу

**Завдання 4**
python task4/task4.py
